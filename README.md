Spring Boot - 2.0.4.RELEASE
JDK - 1.8或更高版本
Spring Framework - 5.0.8 RELEASE
Hibernate - 5.2.17。最后
Maven - 3.2+
IDE - Eclipse或Spring Tool Suite（STS）
Tomcat - 8.5+
Thymeleaf - 3.0.9 RELEASE
Bootstrap - 3.3.7
JQuery - 3.2.1
MySQL - 5.1.46
